📌 Project Overview
Real-time object detection and tracking system built for CodeAlpha AI Internship Task 4.

✨ Features
✅ Real-time video input (webcam)
✅ YOLOv8 pre-trained model (80+ object classes)
✅ Bounding boxes with labels
✅ Object tracking with unique IDs
✅ Movement trails showing object path
✅ Fullscreen mode support
✅ Screenshot capture

🛠️ Technologies Used
Python 3.x
YOLOv8 (Ultralytics)
OpenCV
NumPy

🚀 Installation
pip install ultralytics opencv-python numpy
